Usage:
pinlog <full pe path> <pe name> <log file name>
Open program and take action, Exit program
Log file will be created
Open IDA load Logfile

Example:
pinlog c:\windows\system32\notepad.exe notepad.exe notepad.log

PinTrace is pin32\trace.dll source, to compile with VS2010:

* Extract & Copy into \pin-2.xx-xxxx-msvc10-windows\source\tools
* Copy PinTrace.dll to pin32 folder and rename to trace.dll
